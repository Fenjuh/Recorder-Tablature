MuseScore plugin, that let you add recorder fingerings in your sheets.
(checked on versions 2.3.4 and 3.5.0)

Installation.

1. (2.x.x) Copy files "Recorder_Tablature_S_B.qml", "Recorder_Tablature_S_G.qml", "Recorder_Tablature_A_B.qml", "Recorder_Tablature_A_G.qml" to foler "%userprofile%\Documents\MuseScore2\Plugins" or "%programfiles(x86)%\MuseScore2\Plugins"
1. (3.x.x) Copy files "Recorder_Tablature_S_B3.qml", "Recorder_Tablature_S_G3.qml", "Recorder_Tablature_A_B3.qml", "Recorder_Tablature_A_G3.qml" to foler "%userprofile%\Documents\MuseScore2\Plugins" or "%programfiles(x86)%\MuseScore2\Plugins"
2. Install font "RecorderFontYamahaEdition.ttf" (double click on file - Install)
3. Run MuseScore
4. Go to menu Plugins - Plugin Manager, check Recorder_Tablature_S_B, Recorder_Tablature_S_G, Recorder_Tablature_A_B, Recorder_Tablature_A_G or only what you need, press Îê.
5. In Plugins menu choose Recorder Tablature and needed type. Set font size (from 15 to 60) and press Îê.
6. Enjoy!

7. To remove fingering instant after adding (e.g. to change font size) go to menu Edit - Undo (Ctrl-Z).
8. MuseScore have useful plugin "notenames". Try it.
